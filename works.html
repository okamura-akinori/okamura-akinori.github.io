ポートフォリオサイト

就職活動に向けて、自身の制作物とスキルを 分かりやすく伝えることを目的に制作した 個人ポートフォリオサイトです。

  • 制作目的： 採用担当者が短時間で制作内容を把握できる構成を作るため
  • 課題： 初見でも迷わず作品にたどり着ける導線設計
  • 工夫した点： 1ページ目に作品一覧を配置し、 3クリック以内で詳細が確認できる構成にしました
  • 使用技術： HTML / CSS / GitHub Pages

結果： シンプルな構成にすることで、 スマートフォンからでも 内容が把握しやすいサイトになりました。

今後の改善点： JavaScript を用いた動きのあるUIや、 複数作品を想定したページ構成への拡張を検討しています。
