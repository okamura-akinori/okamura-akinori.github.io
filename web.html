旅行会社のツアーLP

  • 制作時期：2025年12月
  • 制作期間：2週間
  • 使用ソフト：Illustrator/Photoshop
  • 使用技術：HTML / CSS / JavaScript
  • ターゲット：40代女性
  • 目的：「ブランドイメージアップ」が主目的。
    だけど “イメージだけで終わらず→最終的に予約につなげる”

本作品は、気持ちが沈んでいる人や、理由のない疲れを抱えている人に向けた旅のWebデザインです。 多くの旅行広告が「元気」「刺激」「非日常」を前面に出す中で、あえて励まさず、前向きにさせない表現を選びました。 ロンドンという街が持つ曇天や長い歴史、静かな空気感を活かし、「何かを変える旅」ではなく「立ち止まることを許す旅」として設計しています。 コピーは感情を言語化しすぎず、余白を残すことで、見る人が自分の気持ちを重ねられる構成にしました。

IT特化型放課後等デイサービスHP

  • 制作時期：2025年11月
  • 制作期間：2週間
  • 使用ソフト：Illustrator/Photoshop
  • 使用技術：HTML / CSS / JavaScript
  • ターゲット：高機能自閉症等の障害を抱える中高生
  • 目的：問い合わせ数の増加

本作品は、低年齢向けの表現が主流となっている放課後等デイサービス市場に対し、 子ども自身が「選びたくなる」ことを目的としたWebデザインです。 多くの施設サイトが「やさしさ」「安心感」「かわいさ」を前面に出す中で、 本サイトではあえて「かっこよさ」や「憧れ」を軸にした表現を採用しました。 プログラミングやデザインといった創作活動を通じて、 「支援される場所」ではなく 「自分で選び、挑戦する場所」として認識されるよう設計しています。 デザインは子ども扱いしすぎないトーンを意識し、 言葉やビジュアルに余白を持たせることで、 子どもが自分なりの意味や可能性を重ねられる構成としました。
